FINAL PROJECT DEVELOPMENT TOOL
===
THIS SOFTWARE IS FOR MY FINAL PROJECT

EASY FOR DEBUG AND TEST

- FILE
    - OPEN
        - FOLDER
        開啟資料夾內所有圖片並存放到ListBox
        - FILE
        開啟單張圖片
    - SAVE
    儲存處理後的圖片
    - EXIT
    
- DETECTION
    - VEHICLE
    觀測車子的動態變化
    - SPEED
    車速偵測

- EXPORT
    - EXCEL
    輸出資料至EXCEL
    - GRAY
    將圖片轉成灰階並存到測試資料目錄內的GRAY目錄，假如GRAY目錄不存在會自動創建

- LISTBOX FUNCTION
    - 假如ListBox內有選項（圖片路徑），則可以點擊選項直接查看圖片和處理後的圖片
